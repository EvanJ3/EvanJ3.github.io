Multi-class classification for MNIST Comparison¶

The purpose of this project is to compare different classifiers and their performance for multi-class classifications on the complete MNIST dataset at http://yann.lecun.com/exdb/mnist/. The MNIST database of handwritten digits has a training set of 60,000 examples, and a test set of 10,000 examples. We will compare KNN, logistic regression, SVM, kernel SVM, and neural networks.

  • All data will be standardized according to the features before training the classifiers, by dividing the values of the features by 255 (thus map the range of the features from [0, 255] to [0, 1]).

  • For kernel SVM, we will use a radial basis function kernel, and a heuristic called median trick: choose the parameter of the kernel $K(x, x') = \exp\{-\|x-x'\|^2/(2\sigma^2)\}$. Choose the bandwidth as $\sigma=\sqrt{M/2}$ where $M =$ the median of $\{\|x^i-x^j\|^2, 1\leq i, j \leq m', i\neq j \}$ for pairs of training samples. Here we can randomly choose $m'=1000$ samples from training data to use for the median trick.

  • For KNN and SVM, we will randomly downsample the training data to size $m=5000$, to improve computation efficiency.

For each classifier we will train the classifiers on training dataset and evaluate on the test dataset.

Below we begin the problem by preparing the data and training each of the models defined above. The models themselves are trained and saved to the folder using joblib for model persistence. Each of the models I have implemented in this report have been pre-trained and stored.

def down_sample(data,labels,num_observations,seed=None):
    assert data.shape[0] == labels.shape[0]
    if seed != None:
        np.random.RandomState.seed(seed)
    else:
        pass
    indexes = np.random.choice(data.shape[0],num_observations)
    down_sampled_data = data[indexes,:]
    down_sampled_labels = labels[indexes]
    return down_sampled_data, down_sampled_labels



def median_trick(data,num_sample_points,seed=None):
    if seed != None: 
        np.random.Randomstate.seed(seed)
    else:
        pass
    samples = data[np.random.choice(data.shape[0],num_sample_points),:]
    distances = scipy.spatial.distance.pdist(samples,'sqeuclidean')
    median_distance = np.median(distances)
    gamma = 1/(median_distance/2)
    return gamma


def model_reports_dataframe(skl_model,skl_test_data,skl_test_labels,model_prefix):
    skl_model = joblib.load(skl_model)
    predictions = skl_model.predict(skl_test_data)
    class_repo = classification_report(y_true=skl_test_labels,y_pred=predictions,digits=4,output_dict=True)
    class_repo.pop('macro avg')
    class_repo.pop('weighted avg')
    class_repo.pop('accuracy')
    df = pd.DataFrame(class_repo)
    df = df.T.drop(columns=['support'])
    df = df.rename(columns={'precision': model_prefix + ' ' + 'Precision','recall': model_prefix + ' ' + 'Recall','f1-score':model_prefix + ' ' + 'F1-Score'})
    return df
