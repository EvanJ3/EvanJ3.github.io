Welcome to Bridging the Gap, Team 54's project for CSE6242.

This application, developed by Team054 for the Georgia Institute of Technology's CSE 6242: Data and Visual Analytics course, is a modern adaptation of existing tools for understanding and visualizing bridge infrastructure data in the United States.

There are over 617,000 bridges in the country, all of them critical to the transporation of goods, services, and individuals. This application allows anyone to view essential details about each of these critical pieces of infrastructure. Use the navigation bar or links below to explore our interactive dashboards, modeling information, and to learn various information about the project.
